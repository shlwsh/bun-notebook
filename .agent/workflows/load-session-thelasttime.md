---

这个工作流用于帮助 AI 快速回溯并加载上一次开发会话的上下文，确保开发逻辑的连贯性。

1. **定位最新记录**：运行脚本寻找 `sessions/` 目录中时间戳最近的 `.md` 文件。
// turbo
2. **读取会话内容**：获取文件路径后，读取其内容以了解之前的修改点、遗留问题及技术决策。

```bash
# 执行此命令获取最新会话文件路径
bun scripts/get-last-session.ts
```

3. **同步上下文**：AI 将根据读取到的内容调整当前的开发策略，确保与之前的成果无缝衔接。
---
*提示：如果这是您在新会话中第一次与我交流，执行此工作流能让我立即“记起”我们之前做到了哪里。*
