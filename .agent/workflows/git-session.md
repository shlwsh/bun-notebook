---
description: 整理当前 AI 会话成果并同步至远程仓库
---

这个工作流将整理当前的开发进展，生成会话文档并自动提交到远程仓库。

1. **整理会话内容**：总结本次会话的主要修改点和达成目标。
2. **生成会话文档**：将总结内容转存为 `sessions/session_{timestamp}.md` 文件。
// turbo
3. **执行同步提交**：运行以下命令完成文档创建与 Git 提交推送。

```bash
bun scripts/session-sync.ts "在这里替换为本次会话的简短总结" "在这里替换为详细的修改记录"
```

*注意：执行此工作流前请确保已完成所有代码修改并在终端验证通过。*
