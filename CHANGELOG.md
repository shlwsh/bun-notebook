## [0.1.0] - 2026-02-06

### 新增
- 🎉 首次发布 MD Notebook
- 📝 支持 Markdown 文件的查看和编辑
- 💻 支持 TypeScript、JavaScript、TXT 等多种文件类型
- 🎨 集成 CodeMirror 6 编辑器，支持语法高亮
- 🛠️ Markdown 富文本工具栏，支持快捷格式化
- 📊 支持 Mermaid 图表渲染
- 💾 支持导出为 HTML、PDF、DOCX 格式
- 🌍 支持 6 种语言界面（中文、英文、日语、俄语、法语、德语）
- 🎨 支持深色/浅色主题切换
- 📂 文件树浏览和多标签页管理
- 🖼️ 支持图片拖拽和粘贴，自动管理附件目录

### 技术栈
- 前端：Vue 3 + TypeScript + Vite + Tailwind CSS
- 后端：Tauri 2.0 + Rust
- 编辑器：CodeMirror 6
- 包管理：Bun

### 平台支持
- macOS (Intel 和 Apple Silicon)
- Windows (x64)
- Linux (x64)
