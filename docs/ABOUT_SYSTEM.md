# 关于 MD Notebook

## 系统简介

MD Notebook 是一款专业的 Markdown 文档编辑与预览工具，基于 Tauri 2.0、Rust 和 Vue 3 构建，旨在为开发者和文档编写者提供高效、流畅的文档编辑体验。

## 主要功能

### 1. Markdown 编辑
- 专业级 CodeMirror 6 编辑器
- 语法高亮和实时预览
- 智能语法检查和修复建议
- 表格自动补全（回车键）

### 2. Mermaid 图表支持
- 内置 Mermaid 渲染引擎
- 实时预览流程图、时序图等
- 支持多种图表类型

### 3. 文件管理
- 树形文件浏览器
- 多标签页文件管理
- 最近打开项目历史
- 面包屑导航

### 4. 文档导出
- 导出为 HTML 格式
- 导出为 PDF 格式
- 导出为 DOCX 格式
- 保持格式和样式

### 5. 图片处理
- 拖拽插入图片
- 粘贴插入图片
- 自动保存到 attachments 目录
- 自动生成 Markdown 引用

### 6. 主题定制
- 深色/浅色主题切换
- 多种预览主题（默认、浅色、复古、GitHub）
- 可调节字体大小
- 自定义编辑器配置

## 使用说明

### 快速开始

1. **打开项目**
   - 点击顶部工具栏的文件夹图标
   - 或使用菜单：文件 → 打开文件夹
   - 快捷键：`Ctrl+O`（Windows/Linux）或 `Cmd+O`（macOS）

2. **编辑文档**
   - 在左侧文件树中点击 `.md` 文件打开
   - 点击"预览"/"编辑"按钮切换模式
   - 在编辑模式下修改内容

3. **保存文件**
   - 点击顶部的"保存"按钮
   - 或使用快捷键：`Ctrl+S`（Windows/Linux）或 `Cmd+S`（macOS）
   - 修改后标签页会显示保存按钮

4. **导出文档**
   - 点击"导出"按钮
   - 选择导出格式（HTML/PDF/DOCX）
   - 选择保存位置
   - 等待导出完成

5. **插入图片**
   - 在编辑模式下，直接拖拽图片到编辑器
   - 或使用 `Ctrl+V` 粘贴剪贴板中的图片
   - 图片会自动保存到当前文档目录的 `attachments` 文件夹
   - 自动插入 Markdown 图片引用语法

### 快捷键

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 打开文件夹 | `Ctrl+O` | `Cmd+O` |
| 保存文件 | `Ctrl+S` | `Cmd+S` |
| 切换侧边栏 | `Ctrl+B` | `Cmd+B` |
| 表格补全 | `Enter` | `Enter` |
| 退出应用 | `Alt+F4` | `Cmd+Q` |

### 界面说明

- **顶部工具栏**：快速访问常用功能（打开项目、主题切换、设置、关于、退出）
- **左侧活动栏**：切换不同的视图（文件浏览、搜索等）
- **主侧边栏**：显示文件树或搜索结果
- **编辑器区域**：查看和编辑文件内容，支持多标签页
- **底部状态栏**：显示项目信息、版本号、语言设置等

### 高级功能

#### Mermaid 图表

在 Markdown 中使用以下语法创建图表：

\`\`\`mermaid
graph TD
    A[开始] --> B[处理]
    B --> C[结束]
\`\`\`

支持的图表类型：
- 流程图（graph/flowchart）
- 时序图（sequenceDiagram）
- 甘特图（gantt）
- 类图（classDiagram）
- 状态图（stateDiagram）
- 饼图（pie）

#### 表格编辑

在表格行内按 `Enter` 键，自动补全下一行：

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 | ← 按 Enter
```

#### 语法检查

编辑器会自动检查常见的 Markdown 语法问题：
- 标题符号后缺少空格
- 列表格式不正确
- 链接格式错误

点击提示可快速修复。

## 技术架构

### 前端技术栈
- **框架**：Vue 3 + TypeScript
- **状态管理**：Pinia
- **样式**：Tailwind CSS
- **构建工具**：Vite
- **包管理器**：Bun

### 桌面应用
- **框架**：Tauri 2.0
- **后端语言**：Rust
- **跨平台支持**：Windows、macOS、Linux

### 编辑器组件
- **编辑器核心**：CodeMirror 6
- **语法高亮**：Highlight.js
- **Markdown 渲染**：Marked
- **图表渲染**：Mermaid
- **PDF 导出**：html2pdf.js
