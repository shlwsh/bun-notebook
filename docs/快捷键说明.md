# 快捷键说明

## 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd + O` | 打开文件夹 | 选择包含文件的目录 |
| `Ctrl/Cmd + S` | 保存文件 | 保存当前编辑的文件 |
| `Ctrl/Cmd + B` | 切换侧边栏 | 显示/隐藏左侧文件树 |
| `Ctrl/Cmd + W` | 关闭标签页 | 关闭当前打开的文件标签 |

## Markdown 编辑快捷键

### 文本格式化

| 快捷键 | 功能 | Markdown 语法 |
|--------|------|---------------|
| 工具栏 - 粗体 | 粗体文本 | `**文本**` |
| 工具栏 - 斜体 | 斜体文本 | `*文本*` |
| 工具栏 - 删除线 | 删除线文本 | `~~文本~~` |

### 标题

| 操作 | 功能 | Markdown 语法 |
|------|------|---------------|
| 工具栏 - 标题 H1 | 一级标题 | `# 标题` |
| 工具栏 - 标题 H2 | 二级标题 | `## 标题` |
| 工具栏 - 标题 H3 | 三级标题 | `### 标题` |
| 工具栏 - 标题 H4 | 四级标题 | `#### 标题` |
| 工具栏 - 标题 H5 | 五级标题 | `##### 标题` |
| 工具栏 - 标题 H6 | 六级标题 | `###### 标题` |

### 列表

| 操作 | 功能 | Markdown 语法 |
|------|------|---------------|
| 工具栏 - 无序列表 | 无序列表 | `- 项目` |
| 工具栏 - 有序列表 | 有序列表 | `1. 项目` |
| `Enter` (在表格中) | 自动补全表格行 | 自动生成新行 |

### 插入元素

| 操作 | 功能 | Markdown 语法 |
|------|------|---------------|
| 工具栏 - 表格 | 插入表格 | 3x3 表格模板 |
| 工具栏 - 链接 | 插入链接 | `[文本](URL)` |
| 工具栏 - 图片 | 插入图片 | `![描述](路径)` |
| 工具栏 - 代码块 | 插入代码块 | ` ```语言\n代码\n``` ` |
| 工具栏 - 引用 | 插入引用 | `> 引用文本` |
| 拖拽图片 | 插入图片 | 自动保存到 attachments 目录 |
| 粘贴图片 | 插入图片 | 自动保存到 attachments 目录 |

## 编辑器快捷键

### 通用编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Shift + Z` | 重做 |
| `Ctrl/Cmd + C` | 复制 |
| `Ctrl/Cmd + X` | 剪切 |
| `Ctrl/Cmd + V` | 粘贴 |
| `Ctrl/Cmd + A` | 全选 |
| `Ctrl/Cmd + F` | 查找 |
| `Ctrl/Cmd + H` | 替换 |

### 代码编辑 (TypeScript/JavaScript)

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 缩进 |
| `Shift + Tab` | 取消缩进 |
| `Ctrl/Cmd + /` | 切换注释 |
| `Ctrl/Cmd + ]` | 增加缩进 |
| `Ctrl/Cmd + [` | 减少缩进 |

## 文件操作

| 操作 | 功能 |
|------|------|
| 左键单击文件 | 打开文件 |
| 右键单击文件 | 显示上下文菜单 |
| 右键单击标签页 | 关闭选项菜单 |

## 视图操作

| 操作 | 功能 |
|------|------|
| 预览/编辑切换 | 切换 Markdown 预览和编辑模式 |
| 主题按钮 | 切换预览主题 |
| 导出按钮 | 导出为 HTML/PDF/DOCX |
| 打印按钮 | 打印当前文档 |

## 提示

1. **Markdown 工具栏**仅在编辑 Markdown 文件时显示
2. **表格自动补全**：在表格行末按 `Enter` 自动生成新行
3. **图片插入**：支持拖拽和粘贴，自动管理附件目录
4. **语法检查**：Markdown 编辑器会实时检查语法问题
5. **多文件编辑**：可同时打开多个文件，使用标签页切换

## 平台差异

- **Windows/Linux**: 使用 `Ctrl` 键
- **macOS**: 使用 `Cmd` 键

## 自定义快捷键

目前版本暂不支持自定义快捷键，后续版本会添加此功能。
